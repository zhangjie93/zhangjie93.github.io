第33题 Search in Rotated Sorted Array

问题

在一个排序的数组中，如{1,2,3,4,5,6,7}，经过旋转后得到{4,5,6,7,1,2,3}，当然也可以得到原数组{1,2,3,4,5,6,7}，在该旋转后的数组中查找某个元素

解答

在一个排序数组中查找，通常是用二分查找，因为通过中间数字和target能够确定target在数组的哪一边。

在旋转数组中，中间数字的一边是有序，而另一边可能是无序的，在和target比较时，可以确定target是否在有序的那个半边。这样可以缩小target的查找范围

比如{4,5,6,7,1,2,3}，初始nums[mid]是7，此时0~mid是有序的，因为nums[mid]>nums[right]，若target比7小且比4大，则确定target在0~mid之间，比如target是5；若不满足这个条件，则说明target在mid的右边。这样不断的二分，可以找到target

二分查找，在查找时，先确定mid在左半有序区还有右半有序区

int findData(int[] nums,int target){
    int left=0,right=nums.length-1;
    while (left<=right){
        int mid=(right-left)/2+left;
        if(nums[mid]==target){
            return mid;
        }
        //mid在右半有序区
        if(nums[mid]<nums[right]){
            //右半边是有序的
            if(target>nums[mid]&&target<=nums[right]){
                //target在右半边，在mid的右边
                //注意仅有target比nums[mid]大无法判断target在右半边
                left=mid+1;
            }else{
                //此时target会出现两种情况：在右半有序区，但比mid所在所元素小；在左半有序区
                //在右半边
                right=mid-1;
            }
        }else{
            //mid在左半有序区
            //此时target比mid所在元素小，且比left大，则表明target在left和mid之间
            if(target>=nums[left]&&target<nums[mid]){
                right=mid-1;
            }else{
                //target在右半有序区或者target比mid大，则在mid到right之间
                left=mid+1;
            }
        }
    }
    return -1;
}

递归实现

private static int search(int[] arr, int target, int left, int right){
    // 在search失败情况下递归终止的条件
    if (left > right){
        return -1;
    }
    int mid  = (left+right)/2;

    // 左半区有序
    if (arr[mid] >= arr[left]){
        if (target == arr[mid]){
            return mid;
        }
        // target就是在左半区
        if (target >= arr[left] && target < arr[mid]) {
            // 搜索左半区
            return search(arr, target, left, mid-1);
        }else {
            // 搜索右半区
            return search(arr, target, mid+1, right);
        }
    }else {
        // 右半区有序
        if (target == arr[mid]) {
            return mid;
        }
        // target在右半区
        if (target <= arr[right] && target > arr[mid]){
            return search(arr, target, mid+1, right);
        }else {
            return search(arr, target, left, mid-1);
        }
    }
}
