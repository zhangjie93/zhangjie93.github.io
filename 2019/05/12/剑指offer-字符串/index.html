替换空格

将一个字符串中的空格替换成 “%20”。

先遍历字符串，遇到空格则在字符串尾部填充任意字符，使得字符串的长度等于替换之后的长度。因为一个空格要替换成三个字符（%20），因此当遍历到一个空格时，需要在尾部填充两个任意字符。

令 P1 指向字符串原来的末尾位置，P2 指向字符串现在的末尾位置。P1 和 P2 从后向前遍历，当 P1 遍历到一个空格时，就需要令 P2 指向的位置依次填充 0、2、%（注意是逆序的），否则就填充上 P1 指向字符的值。

从后向前遍是为了在改变 P2 所指向的内容时，不会影响到 P1 遍历原来字符串的内容。

public String replaceSpace(StringBuilder str){
    //p1指向原字符串的末尾
    int p1=str.length()-1;
    for (int i = 0; i <=p1; i++) {
        if(str.charAt(i)==' '){
            //有一个空格在末尾加两个字符，结束后的字符串长度就是替换完空格后字符串应有的长度
            str.append("  ");
            //这里用两个空格来占位，可以用任意字符占位
        }
    }
    //如：字符串"A B C D"，在上面填充完成后就是A B C D      ",在后面加了6个空格
    //p1指向D，p2指向最后一个空格因为p1指向不是空格，所以p2指向的位置填充上D。p1，p2前进。
    //p1此时指向空格，所以p2指向的地方放入0，p2前进，放入2，p2前进，放入%，p2前进，此时p1也要前进，依次类推
    //p2是现在字符串末尾的位置
    int p2=str.length()-1;
    while (p1>=0&&p2>p1){
        char c=str.charAt(p1--);
        if(c==' '){
            //为空格,则填充02%，注意是%20的逆序
            str.setCharAt(p2--,'0');
            str.setCharAt(p2--,'2');
            str.setCharAt(p2--,'%');
        }else{
            str.setCharAt(p2--,c);
        }
    }
    return str.toString();
}

字符串转整数

将一个字符串转换成一个整数(实现Integer.valueOf(string)的功能，但是string不符合数字要求时返回0)，要求不能使用字符串转换整数的库函数。 数值为0或者字符串不是一个合法的数值则返回0

解答：只需要按字符判断即可，对于第一个字符可以为数字或者正负号，其它位置的字符都应该是数字。在遍历时判断是否合法即可

public int StrToInt(String str) {
    if(str==null||str.length()<=0){
        return 0;
    }
    //正负号
    int flag=1;
    flag=str.charAt(0)=='-'?-1:1;
    int res=0;
    for (int i = 0; i < str.length(); i++) {
        char c=str.charAt(i);
        //第1个字符单独判断
        if(i==0&&(c=='+'||c=='-')){
            continue;
        }
        //合法
        if(c>='0'&&c<='9'){
            res=res*10+(c-'0');
        }else{
            return 0;
        }
    }
    return flag*res;
}

表示数值的字符串

请实现一个函数用来判断字符串是否表示数值（包括整数和小数）。例如，字符串”+100”,“5e2”,“-123”,“3.1416”和”-1E-16”都表示数值。 但是”12e”,“1a3.14”,“1.2.3”,“+-5”和”12e+4.3”都不是

解答：

这题可以用正则[+-]?\\d*(\\.\\d+)?([eE][+-]?\\d+)?，判断是否符合该正则。这个正则有问题e12也能通过

正则解释：[+-]?表示有0或1个+/-，然后是0或多个数字，有0个或1个小数点+数字，然后是可选的e/E，+/-可选（指0或1），后面是至少一个数字

按位判断

public boolean isNumeric(char[] str) {
    boolean sign=false,dot=false,hasE=false;
    for (int i = 0; i < str.length; i++) {
        if(str[i]=='e'||str[i]=='E'){
            //指数符号前必须有整数
            if(i==0){
                return false;
            }
            //e不能是最后一位
            if(i==str.length-1){
                return false;
            }
            //不能有两个e
            if(hasE){
                return false;
            }
                hasE=true;
        }else if(str[i]=='+'||str[i]=='-'){
            //第一次出现+''-'符号只能在第一个字符或者指数符号后
            if(!sign && i != 0 && !hasE)  {
                return false;
            }
            //第二次出现一定要在e/E的后面
            if(sign&&str[i-1]!= 'e' && str[i-1] != 'E'){
                return false;
            }
            //第一次出现不在字符串开头则必须在e/E后面
            if(!sign&&i>0&&str[i-1]!='e'&&str[i-1]!='E'){
               return false;
            }
            sign=true;
        }else if(str[i]=='.'){
            // // e后面不能接小数点，小数点不能出现两次
            if(hasE||dot){
                return false;
            }
            dot=true;
        } else if (str[i] < '0' || str[i] > '9') // 不合法字符
            return false;
    }
    return true;
}

字符流中第一个不重复的字符

请实现一个函数用来找出字符流中第一个只出现一次的字符。例如，当从字符流中只读出前两个字符 “go” 时，第一个只出现一次的字符是 “g”。当从该字符流中读出前六个字符“google” 时，第一个只出现一次的字符是 “l”。

如果当前字符流没有存在出现一次的字符，返回#字符。

注意是数据流，所以数据的数目是不断变化的

用数组记录次数，队列中只保留次数为1的字符

public class FirstAppearingOnce {
    private int[] cnts = new int[256];
    private Queue<Character> queue = new LinkedList<>();

    //Insert one char from stringstream
    public void Insert(char ch) {
        queue.add(ch);
        //记录出现次数
        cnts[ch]++;
        //去掉大于1的
        while (!queue.isEmpty()&&cnts[queue.peek()]>1){
            queue.poll();
        }
    }

    //return the first appearence once char in current stringstream
    public char FirstAppearingOnce() {
        return queue.isEmpty() ? '#' : queue.peek();
    }
}

把数字翻译成字符串

给定一个数字，按照如下规则翻译成字符串：1 翻译成“a”，2 翻译成“b”… 26 翻译成“z”。一个数字有多种翻译可能，例如 12258 一共有 5 种，分别是 abbeh，lbeh，aveh，abyh，lyh。实现一个函数，用来计算一个数字有多少种不同的翻译方法

解答：

public class NumDecodings {
    public int numDecodings(String s) {
        if (s == null || s.length() == 0 || s.charAt(0) == '0') {
            return 0;
        }
        int len=s.length();
        int[] dp=new int[len+1];
        dp[0]=0;
        dp[1]=s.charAt(0)=='0'?0:1;
        for (int i = 2; i <=len; i++) {
            //取第i个字符
            char c=s.charAt(i-1);
            //若不为0则可以将c翻译
            if(c!='0'){
                dp[i]+=dp[i-1];
            }
            //如果前一个字符为0，则c和不能为前一个字符组成一个数
            if(s.charAt(i-2)=='0'){
                continue;
            }
            //第i个字符和第i-1个字符组成一个数字
            if(Integer.parseInt(s.substring(i-2,i))<=26){
                //在前两位数的情况下，如123，在i=2时，12是符合情况的，此时需要加1
                if(i-2==0){
                    dp[i]+=1;
                }else{
                    dp[i]+=dp[i-2];
                }


            }

        }
        return dp[len];
    }
}

第一个只出现一次的字符位置

在一个字符串(0<=字符串长度<=10000，全部由字母组成)中找到第一个只出现一次的字符,并返回它的位置, 如果没有则返回 -1（需要区分大小写）

解答：

遍历字符串，将每个字符的个数统计下来，然后再次遍历字符串，找到个数为1的字符

public int FirstNotRepeatingChar(String str) {
    Map<Character,Integer> map=new HashMap<>();
    //map存储字符个数
    for (int i = 0; i < str.length(); i++) {
        char c=str.charAt(i);
        if(map.containsKey(c)){
            map.put(c,map.get(c)+1);
        }else{
            map.put(c,1);
        }
    }
    //重新遍历字符串，得到个数为1的字符
    for (int i = 0; i < str.length(); i++) {
        char c=str.charAt(i);
        if(map.get(c)==1){
            return i;
        }
    }
    return -1;
}

由于字符是有限的，所以可以用数组代替HashMap

public int FirstNotRepeatingChar1(String str){
    //由于字符的范围是固定的，所以用数组来代替HashMap
    //索引就表示字符对应的ASCII
    int[] cnt=new int[256];
    for (int i = 0; i < str.length(); i++) {
        char c=str.charAt(i);
        cnt[c]++;
    }
    for (int i = 0; i < str.length(); i++) {
        char c=str.charAt(i);
        if(cnt[c]==1){
            return i;
        }
    }
    return -1;
}

在计数时，其实只需记录字符数是0，1以及不是0或1，所以一个字符只需要2个bit位就可以记录

public int FirstNotRepeatingChar2(String str) {
    BitSet bs1 = new BitSet(256);
    BitSet bs2 = new BitSet(256);
    for (char c : str.toCharArray()) {
        //如果是第一次遇到，则将第1个比特位(从右向左)设为1，表示字符数为1
        if (!bs1.get(c) && !bs2.get(c))
            bs1.set(c);     // 0 0 -> 0 1
        //如果第二次遇到，则将第2个比特位设为1，表法字符数已经大于1了
        else if (bs1.get(c) && !bs2.get(c))
            bs2.set(c);     // 0 1 -> 1 1
    }
    for (int i = 0; i < str.length(); i++) {
        char c = str.charAt(i);
        //只有第1个比特位为1的字符个数才是1
        if (bs1.get(c) && !bs2.get(c))  // 0 1
            return i;
    }
    return -1;
}

翻转单词顺序列

牛客最近来了一个新员工Fish，每天早晨总是会拿着一本英文杂志，写些句子在本子上。同事Cat对Fish写的内容颇感兴趣，有一天他向Fish借来翻看，但却读不懂它的意思。例如，“student. a am I”。后来才意识到，这家伙原来把句子单词的顺序翻转了，正确的句子应该是“I am a student.”。Cat对一一的翻转这些单词顺序可不在行，你能帮助他么？

Input:
"I am a student."

Output:
"student. a am I"

解答

先旋转每个单词，再旋转整个字符串

public class ReverseSentence {
    public String ReverseSentence(String str) {
        int n=str.length();
        char[] c=str.toCharArray();
        int j=0;
        for (int i = 0; i <=n; i++) {
            if(i==n||c[i]==' '){
                reverse(c,j,i-1);
                j=i+1;
            }
        }
        reverse(c,0,n-1);
        return new String(c);
    }
    //翻转i到j
    private void reverse(char[] c, int i, int j) {
        while (i < j)
            swap(c, i++, j--);
    }
    private void swap(char[] c, int i, int j) {
        char t = c[i];
        c[i] = c[j];
        c[j] = t;
    }

}

左旋转字符串

汇编语言中有一种移位指令叫做循环左移（ROL），现在有个简单的任务，就是用字符串模拟这个指令的运算结果。对于一个给定的字符序列S，请你把其循环左移K位后的序列输出。例如，字符序列S=”abcXYZdef”,要求输出循环左移3位后的结果，即“XYZdefabc”。是不是很简单？OK，搞定它！

Input:
S="abcXYZdef"
K=3

Output:
"XYZdefabc"

先将 “abc” 和 “XYZdef” 分别翻转，得到 “cbafedZYX”，然后再把整个字符串翻转得到 “XYZdefabc”

public String LeftRotateString(String str, int n) {
    if (n >= str.length())
        return str;
    char[] chars = str.toCharArray();
    reverse(chars, 0, n - 1);
    reverse(chars, n, chars.length - 1);
    reverse(chars, 0, chars.length - 1);
    return new String(chars);
}

private void reverse(char[] chars, int i, int j) {
    while (i < j)
        swap(chars, i++, j--);
}

private void swap(char[] chars, int i, int j) {
    char t = chars[i];
    chars[i] = chars[j];
    chars[j] = t;
}

最长不含重复字符的子字符串

输入一个字符串（只包含 a~z 的字符），求其最长不含重复字符的子字符串的长度。例如对于 arabcacfr，最长不含重复字符的子字符串为 acfr，长度为 4

解答

首先考虑如果手工来做会如何做。以abcabcbb为例，从头开始遍历，首先a加入子串，然后b加入，c加入，当到a时，由于前面的字符串已经有a，所以子串变为bca，之前符合条件的子串长度为3。依次遍历下去，最后得到最长的不重复子串的长度为3

这个过程就像一个滑动窗口，初始时滑动窗口为空，然后遍历字符串，第一个字符a加入滑动窗口，然后b加入，c加入，当到a时，由于滑动窗口中已有a，所以滑动窗口的左边界需要指向前面重复元素的下一个元素，也就是前一个a的下个元素b，然后右边界需要指向当前的a。

所以当字符在窗口中不存在时加入到滑动窗口，如果已有，则将左边界移到前面重复元素的下一个元素，右边界指向当前的元素

public int lengthLongestSubStr(String s){
    int left=0,right=0;
    if(s.length()==0||s==null){
        return 0;
    }
    int max=0;
    HashMap<Character,Integer> slideWindow=new HashMap<>();
    while (right<s.length()){
        //right指向当前窗口的下一个元素
        char c=s.charAt(right);
        //如果right指向的元素不在窗口内，则加入，且right右移
        if(!slideWindow.containsKey(c)){
            slideWindow.put(c,right);
            right++;
        }else{
            //在窗口内则需要将前面重复的元素
            max=slideWindow.size()>max?slideWindow.size():max;
            int tempLeft=slideWindow.get(c)+1;
            //这里left记录的是窗口的边界，所以需要从左边界删除到重复元素
            while (left<tempLeft){
                slideWindow.remove(s.charAt(left++));
            }
            //right指向的元素要加入到滑动窗口内
            slideWindow.put(c,right);
            right++;
        }
    }
    //最后返回最大值，这里不能直接返回max
    return slideWindow.size()>max?slideWindow.size():max;
}

下面使用了HashSet，思想是一样的，出现的字符放入Set中，left指向左边界。right指向窗口的下一个字符，如果没有重复的，right指向的字符会加入，否则从left开始删除，直到删除到重复字符

public int lengthOfLongestSubstring(String s) {
    int res = 0, left = 0, right = 0;
    HashSet<Character> t = new HashSet<Character>();
    while (right < s.length()) {
        //不含字符，加入
        if (!t.contains(s.charAt(right))) {
            t.add(s.charAt(right++));
            res = Math.max(res, t.size());
        } else {
            //从左边界开始删除，因为删除一个后，如果还没有删除重复字符后，在下一个循环会继续删除，直到删掉重复字符
            //由于这里采用了Set，无法直接取到重复字符的索引，所以无法像上面的代码一样用另一个循环删除
            t.remove(s.charAt(left++));
        }
    }
    return res;
}
