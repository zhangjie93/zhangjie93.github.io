第6题 Zigzag Conversion

问题

给定一个字符串s和一个正整数numRows。将字符串按照“之”字形重写，行数是numRows

如s="PAYPALISHIRING",numRows=3，重写成如下字符串

P   A   H   N
A P L S I I G
Y   I   R

解答

这题就是找规律

把字符串全换成索引，设numRows=5

0           8             16  //第一行数的间隔是numRow2*2-2
1        7  9          15
2     6     10      14
3  5        11  13
4           12  

 public  String fill(int start,int initDistance,int magic,String s){
    StringBuilder sb=new StringBuilder();
    while ((start<s.length())){
        //最后一行和第一行的initial distance应该是magic
        if(initDistance==0) {
            initDistance=magic;
        }
        sb.append(s.charAt(start));
        //加上距离
        start+=initDistance;
        //下一个距离和本次距离之和为magic
        initDistance=magic-initDistance;
    }
    return sb.toString();
}
public String convert(String s, int numRows) {
    int length=s.length();
    if(length==0||numRows==1||s==null){
        return s;
    }
    StringBuilder res=new StringBuilder(s.length());
    //magic表示转换之后第一行字符之间的距离，后面递减2
    int magic=numRows+numRows-2;
    int distance=magic;
    //每一行的初始
    for (int i = 0; i < numRows; i++) {
        res.append(fill(i,distance,magic,s));
        distance-=2;
    }
    return res.toString();
}
