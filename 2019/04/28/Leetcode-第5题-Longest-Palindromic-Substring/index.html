第5题 Longest Palindromic Substring

问题

给定字符串s，在s中找到最长回文子串。s的最长长度是1000.

s="babad"
substring="bab"； //最长回文子串

解答

解法1

假定子串以i为中心，然后向两端扫描，直到不是回文串为止。

public String longestPalindrome(String s) {
    String longString = "";
    int len = s.length();
    //以i为中心
    for (int i = 0; i < len; i++) {
        String temp="";
        //分成子串为奇数和偶数的情况
        //子串为奇数的情况，也就是以s[i]为中心的子串
        for (int j = 0; (i - j) >= 0 && (i + j) < len; j++) {
            if (s.charAt(i-j)!=s.charAt(i+j)){
                break;
            }
            temp=s.substring(i-j,i+j+1);
        }
        if (temp.length()>longString.length()){
            longString=temp;
        }
        for (int j=0;(i-j)>=0&&(i+j+1<len);j++){
            if (s.charAt(i-j)!=s.charAt(i+j+1)){
                break;
            }
            temp=s.substring(i-j,i+j+2);
        }
        if (temp.length()>longString.length()){
            longString=temp;
        }
    }
    return longString;
}

解法2

动态规划

dp[i][j]表示以s[i]和s[j]开头的和结尾的字符串是否是回文串

public String longestPalindrome(String s) {
        if(s.length()<=1){
            return s;
        }
        int dp[][]=new int[s.length()][s.length()];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < s.length(); j++) {
                if (i>=j){
                    dp[i][j]=1;
                }else{
                    dp[i][j]=0;
                }
            }
        }
        int iMin=0;
        int iMax=0;
        int len=1;
//        for (int i = 0; i < s.length()-1; i++) {  //错误代码，如果按行填dp,计算dp[i][j]会出现dp[i+1][j-1]还没有计算的情况
//            for (int j = i+1; j < s.length(); j++) {
//                if(s.charAt(i)!=s.charAt(j)){
//                    dp[i][j]=0;
//
//                }else{
//                    if (dp[i+1][j-1]==1){
//                        dp[i][j]=1;
//                        if(j-i+1>len){
//                            iMin=i;
//                            iMax=j;
//                            len=j-i+1;
//                        }
//                    }else{
//                        dp[i][j]=0;
//                    }
//                }
//            }
//        }
        //k相当于字符串的长度
        for (int k = 1; k < s.length(); k++) {
            for (int i = 0; k+i < s.length(); i++) {
                int j=i+k;
                if(s.charAt(i)!=s.charAt(j)){
                    dp[i][j]=0;
                }else{
                    dp[i][j]=dp[i+1][j-1];
                    if(dp[i][j]==1&&j-i+1>len){
                            iMin=i;
                            iMax=j;
                            len=j-i+ 1;
                    }
                }
            }
        }
        return s.substring(iMin,iMax+1);
    }
