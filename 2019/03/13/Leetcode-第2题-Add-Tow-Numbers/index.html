第2题 Add Two Numbers

问题

给定两个整数，这两个整数是以链表的形式存储的，且是以倒序的形式存储的。如234存储的方式是4->3->2，求这两个数的和，和也是以相同的方式存储。

这两个数都是非负整数

解答

这题是就基本的链表操作，遍历两个链表，遍历的同时执行相加的操作。主要的问题是注意进位。同时，由于两个数的长度可能不同，所以需要处理这种情形。处理的方法主要是加0或者是单独遍历较长的数。

结点的定义

class ListNode {
    int val;
    ListNode next;

    ListNode(int x) {
        val = x;
    }
}

解法1

此方法采用循环的方式，遇到有数字长度不够时，采用加0的方式

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        if (l1 == null || l2 == null) {
            return null;
        }
        ListNode temp = new ListNode(0);
        ListNode result = temp;  //头结点

        int value1 = 0;
        int value2 = 0;
        while (l1 != null && l2 != null) {
            //当前位，计算时需要加上之前的进位
            value2 = (l1.val + l2.val + value1) % 10;
            //进位
            value1 = (l1.val + l2.val + value1) / 10;
            //由于是倒序，所以求出的每一位是放在链表的尾部，temp一直指向尾部节点
            temp.next = new ListNode(value2);
            //移动l1,l2，temp
            l1 = l1.next;
            l2 = l2.next;
            temp = temp.next;
            if (l1 == null && l2 == null) { //两个同时为空，也就是较长的已经结束
                break;
            }
            //l2还有数
            if (l1 == null) {
                l1 = new ListNode(0);  //next默认为null
            }
            //l1还有数
            if (l2 == null) {
                l2 = new ListNode(0);
            }
        }
        if (value1 != 0) {  //最后还有进位
            temp.next = new ListNode(value1);
        }
        return result.next;
    }

解法2

这种方法类似于归并排序中合并的过程，当长度不同时，先将相同长度部分相加，然后单独处理较长的数中剩余的部分

public ListNode addTwoNumbers3(ListNode l1, ListNode l2) {
    if (l1 == null && l2 == null) {
        return null;
    }
    ListNode head = new ListNode(0);
    ListNode point = head;
    int carry = 0; //存储进位
    //处理长度相同的部分
    while (l1 != null && l2 != null) {
        int sum = carry + l1.val + l2.val;
        point.next = new ListNode(sum % 10);
        carry = sum / 10;
        l1 = l1.next;
        l2 = l2.next;
        point = point.next;
    }
    //处理剩余的部分
    //注意之前的进位
    while (l1 != null) {
        int sum = carry + l1.val;
        point.next = new ListNode(sum % 10);
        carry = sum / 10;
        l1 = l1.next;
        point = point.next;
    }
    while (l2 != null) {
        int sum = carry + l2.val;
        point.next = new ListNode(sum % 10);
        carry = sum / 10;
        l2 = l2.next;
        point = point.next;
    }
    //还有最后一个进位
    if (carry != 0) {
        point.next = new ListNode(carry);
    }
    return head.next;
}

解法3

这种方法是采用递归的方式实现。这种递归的写法是将进位单独处理，先计算不进位的，然后单独将进位加上.

public ListNode addTwoNumbers2(ListNode l1, ListNode l2) {
    if (l1 == null || l2 == null) {
        return l1 == null ? l2 : l1;
    }
    int value = l1.val + l2.val;
    //当前位的数
    ListNode result = new ListNode(value % 10);
    //计算后面的结果
    result.next = addTwoNumbers2(l1.next, l2.next);
    if (value >= 10) {
        //有进位，相当于进位和后面的数相加
        result.next = addTwoNumbers2(new ListNode(value / 10), result.next);
    }
    return result;
}

public ListNode helper(ListNode l1, ListNodel2, int carry){
    //结束，需要注意是否有进位
    if(l1==null && l2==null){

        l1 = new ListNode(0);
    }
    if(l2==null && l1!=null){
        l2 = new ListNode(0);
    }

    //当前位
    ListNode curr = new ListNode(sum % 10);
    curr.next = helper(l1.next, l2.next, sum/ 10);
    return curr;
}
